# TimeRabbit for Jira

Jira タスクの作業時間をアクティビティ種別ごとに計測・記録する Chrome 拡張機能。

## 機能

- Jira タスクページ・ボード・バックログを開くと自動検出、ワンクリックで計測開始
- アクティビティ種別（設計 / 実装 / レビュー実施 / レビュー対応 / その他）ごとに記録
- アクティビティを切り替えると現在のセッションが確定し、新しいアクティビティで即再開
- 別タスクに切り替えると前のタイマーを自動停止
- ポップアップから計測中タイマーの開始時刻を後から修正可能
- **手動ログ追加・編集・削除**（タイマー押し忘れ対策）
- ダッシュボードで日別・タスク別・アクティビティ別に集計（グラフ表示）
- **時間軸ビュー**: ログを時系列で表示し、空き時間（ギャップ）をクリックして即追加
- CSV / JSON エクスポート

## インストール

### ビルド

```bash
pnpm install
pnpm build
```

### Chrome に読み込む

1. `chrome://extensions` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」で `dist/` フォルダを選択

## 使い方

### タイマー計測

1. Jira のタスクページ（`/browse/PROJ-123`）、またはボード・バックログで課題パネルを開く
2. Chrome ツールバーの拡張機能アイコンをクリック
3. アクティビティを選択して「計測開始」

ページ右下のフローティングウィジェットからも開始・停止できます（設定で非表示にも可）。

### 手動ログ追加

ダッシュボードのヘッダー「＋ ログを追加」から、任意の日時・タスク・アクティビティを記録できます。

- 終了時刻 / 所要時間のどちらでも入力可能（例: `1h30m`, `90m`, `1:30`）
- 時間軸ビューのギャップ行をクリックすると時刻がプリセットされた状態でモーダルが開く

### ダッシュボード

拡張機能アイコンをクリックし、ヘッダー右上の「↗」からダッシュボードを開く。

| タブ | 内容 |
|---|---|
| タスク軸 | タスク別合計時間ランキング。行クリックでログの編集・削除 |
| 時間軸 | 日付ごとの時系列ログ一覧。ギャップ行クリックで手動追加 |
| タスクキャッシュ | 取得済み Jira メタデータ一覧（Epic・SP・スプリント・実工数） |
| 設定 | Jira ベース URL・フローティングウィジェット表示などを設定 |

### エクスポート

ダッシュボード右上の「エクスポート」から CSV / JSON で出力。

## 対応 URL

- `https://*.atlassian.net/browse/*`
- `https://*.atlassian.net/jira/software/projects/*/issues/*`
- `https://*.atlassian.net/jira/*/issues/*`
- `https://*.atlassian.net/jira/software/projects/*/boards/*`
- `https://*.atlassian.net/jira/software/c/projects/*/boards/*`

## 設定

ダッシュボードの「設定」タブから変更できます。

| 設定項目 | 説明 |
|---|---|
| Jira ベース URL | プロジェクトキーごとに Jira インスタンスの URL を登録（例: `PROJ` → `https://org.atlassian.net`）。「Jiraで開く」リンクの生成に使用 |
| フローティングウィジェット | Jira ページ右下のタイマーカードの表示/非表示 |

### 上級設定（`chrome.storage.local` の `timeRabbit_settings` を直接編集）

| キー | デフォルト | 説明 |
|---|---|---|
| `storyPointsFieldId` | `customfield_10053` | ストーリーポイントのカスタムフィールドID（初回ロード時に自動検出） |
| `sprintFieldId` | `customfield_10020` | スプリントのカスタムフィールドID |
| `metaCacheTtlMs` | `3600000`（1時間） | タスクメタキャッシュの有効期限 |
| `archiveThresholdDays` | `90` | アーカイブ対象とする経過日数 |

## 開発

```bash
pnpm dev       # ウォッチモードでビルド
pnpm build     # プロダクションビルド
pnpm test      # テスト実行
```
